# Лабораторная работа №4: Проектирование и наполнение базы данных

## Описание задачи
1. разработать физическую схему БД (DDL);
2. определить ограничения целостности;
3. подготовить тестовые данные;
4. автоматизировать пересоздание базы данных.


## Используемые технологии
- СУБД: (PostgreSQL)
- Язык генерации тестовых данных: (Python 3.x с библиотеками psycopg2, Faker)
- Среда выполнения: (ОС, необходимые пакеты)

## Состав проекта

- `lab4.sql` – DDL-скрипт создания схемы (таблицы, индексы, ограничения).
- `initial.sql` – начальное наполнение (минимальные справочные данные).
- `gen.py` – скрипт на Python для генерации тестовых данных и загрузки их в БД.
